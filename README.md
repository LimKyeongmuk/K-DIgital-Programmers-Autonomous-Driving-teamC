## Environment & Installation
---
- Ubuntu 18.04
- ROS Melodic
- ar-track-alvar v0.7.1
~~~bash
$ sudo apt update
$ sudo apt-get install ros-melodic-ar-track-alvar
~~~
- pygame v1.9.6
~~~bash
$ sudo apt update
$ sudo apt-get install python-pip
$ pip2 install pygame==1.9.6
~~~
- pillow v6.2.2 
~~~bash
$ pip2 install pillow==6.2.2
~~~
